Voigt.ai

A multi-tenant cloud hosted (K8), application for managing benchmarks and tests of arbitrary models and AI pipelines.

I created React and MaterialUI components for all model evaluation tracking UI and application, designed and developed Charting of model metrics with Recharts.js and D3, worked on model visualization modules with D3.js and shape library to display explanation types. I also built an AWS Lambda backend to implement pipeline execution and lifecycle for automated model benchmarking.

Team Size: 4 people

Tech Stack: Google Cloud, Kubernetes,, Node.js Express.js, React, MaterialUI, D3, Recharts.js

StockTrigger

A conversational app on Amazon Echo, focused on stock quoting and tracking use cases.

StockTrigger relies on Node.js microservices backend, the REST/JSON API layer and API schemas, as well as scalable connectors running in recurring fashion to synchronize stock prices in near realtime with various financial databases and services via AWS Cloudwatch Events, AWS SQS and Lambdas.

Tech Stack: Node.js, Alexa Skills Kit, Express, DynamoDB, AWS Lambda, SQS, Angular.js
